A) Install Plugin - top

Split Test URL Rotator contains a WordPress plugin that must be installed in your blog.

To install it, upload wp-split-test.zip within the WordPress admin panel, just like any other WordPress plugins and activate it.

After the plugin is activated, the plugin can be found under the WordPress Settings menu in the dashboard.

B) Create a New Split Test - top

Before you can create a new split test, you'll need to select or create a new page. Note: Only the URL of this page will be used and none of the content on this page will be shown. It is also advisable to create your landing page or offer pages prior to setup so you don't need to keep going back to the pages menu.

Step 1: Setup the Rotator Page

Go to the plugin. On the plugin's "settings" tab, select the blank page you created previously. (If you didn't create a new page prior to setup, then create a blank page now and then select it here.)

Step 2: Setup the URLs to Multiple Landing Pages or Offer Pages

Click on the "Landing Pages" or the "Offer Pages" tab. Basically they will do the same thing but the separate tabs allows you to separate out different types of split testing that you want to do and keep it organized in its own list.

On either tab (landing pages or offer pages tab), you'll want to enter the following information:

  1. Click "Add Rotation" button.
  2. Name the split test by entering the name into the "Profile" field and enter the number of pages you want to test by entering the number into "Rotations" field.
  3. Enter the URL for the landing pages you want to test into "Landing URL 1", "Landing URL 2", etc. Include http:// in the URL.
  4. Enter the weight that you want to apply to each URL into "Weightable for URL 1", "Weightable for URL 2", etc. (Note: the weight is a percentage of traffic that comes to the page you setup in step 1 that will be redirected to URL 1 or 2 or 3... If you add up all the numbers entered into the "weight" field for all the landing pages or offers they should equal 100.)
  5. Click on the "Add" button.
  6. Click the "Save" button. It should say "Done" and display the URL when it is saved properly. If it still says "Refresh to view" like the screenshot below then it hasn't been saved yet and you'll need to click "save" button to finialize it.

Step 3: Start Testing

Use the URL listed beside the profile name in your advertising to split test the different landing pages or offers that you just setup. If you've just created your split test, then you may need to refresh the page before the URL appears here.

C) View/Stats/Delete Split Tests - top

View the split tests that you've setup on the "Landing Pages" or "Offer Pages" tab. To view the different URLs for each profile, click on the pie chart icon. The statistics for the number of views for each URL can be found here too.

Delete a split test by clicking on the red "x" icon beside the profile name.
